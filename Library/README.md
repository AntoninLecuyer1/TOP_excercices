
markdown# 📚 Library App

A simple book library manager built with vanilla JavaScript.

## Features
- Add books to your library
- Display books as cards
- Mark books as read / unread
- Remove books

## Built with
- HTML5
- CSS3
- Vanilla JavaScript (DOM manipulation, objects)
